# Active Directory Attack & Detection Portfolio

A hands-on research portfolio documenting Active Directory 
attack techniques, detection strategies, and mitigations.

## Lab Environment
- **Domain:** REDTEAM.LOCAL
- **Domain Controller:** AD-DC01 (Windows Server 2022)
- **Attack Platform:** Kali Linux
- **SIEM:** Wazuh

## Detection Approach
Each entry includes:
- Lab setup (GUI and CLI)
- Attack demonstration (Kali Linux and Windows)
- Custom Wazuh detection rules
- MITRE ATT&CK mapping
- Mitigations and remediations

## Tools Used
**Offensive:** Impacket, Mimikatz, Rubeus, NetExec, Evil-WinRM

**Defensive:** Wazuh SIEM, Windows Event Logs, Custom Detection Rules
